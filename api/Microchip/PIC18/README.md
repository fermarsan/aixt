## Description

The API for the PIC18 core includes the modules:

- [pin_fn](Microchip.PIC18.pin_fn.html) for the pin management.
- [pin_fast](Microchip.PIC18.pin_fast.html) for the pin management (faster version).
- [port](Microchip.PIC18.port.html) for the pin port management.
- [timer0](Microchip.PIC18.timer0.html) for the timer0.
- [ext](Microchip.PIC18.ext.html) for the external interrupt through RB0/INT pin.
- [extb](Microchip.PIC18.extb.html) for the external interrupt by changes on RB7-RB4 pins.
