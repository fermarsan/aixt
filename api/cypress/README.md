## Description

`cypress` is not a module by itself, but a namespace for the Cypress devices API modules. Some of functions are shared by most of devices.

The API is split in several folders:


### Compiler related modules

- [psoc_creator](cypress.psoc_creator.time.html)


### Device related modules

- [CY8CKIT_049_42XX](cypress.cy8ckit_049_42xx.pin.html)
- [CY8CKIT_145_40XX](cypress.cy8ckit_145_40xx.pwm.html)
