## Description

`pic24` is not a module by itself, but a namespace for the API modules of Microchip PIC24 target devices. Some of functions are shared by most of targets. The API for the PIC24 includes the core modules and the additional ones.

- [pin_fn](microchip.pic24.pin_fn.html) for the pin management.
- [pin_fast](microchip.pic24.pin_fast.html) for the pin management (faster version).
- [port](microchip.pic24.port.html) for the pin port management.
- [timer0](microchip.pic24.timer0.html) for the timer0.
- [ext](microchip.pic24.ext.html) for the external interrupt.
