## Description

`exp16_pic24` is not a module by itself, but a namespace for the API modules of Microchip Exp16_PIC24 target. Some of functions are shared by most of targets. 


### Exp16_PIC24 modules

- [pic16](microchip.pic24.html) 
- [pin_fn](microchip.exp16_pic24.pin_fn.html) 
- [pin_fast](microchip.exp16_pic24.pin_fast.html) 
- [port](microchip.exp16_pic24.port.html)
- [adc](microchip.exp16_pic24.adc.html)
