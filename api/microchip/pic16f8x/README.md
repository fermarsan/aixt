## Description

`pic16f8x` is not a module by itself, but a namespace for the API modules of Microchip PIC16f8x target devices. Some of functions are shared by most of targets. 


### PIC16F8x modules

- [pic16](microchip.pic16.html) 
- [pin_fn](microchip.pic16f8x.pin_fn.html) 
- [pin_fast](microchip.pic16f8x.pin_fast.html) 
- [port](microchip.pic16f8x.port.html)

### PIC16F8x family targets

  - PIC16F83
  - PIC16F84(A)
