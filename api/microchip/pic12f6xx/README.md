## Description

`pic12f6xx` is not a module by itself, but a namespace for the API modules of Microchip PIC12F6xx target devices. Some of functions are shared by most of targets. 


### PIC12F6xx modules

- [pic12](microchip.pic12.html) PIC12 core modules.
- [pwm](microchip.pic12f6xx.pwm.html) for the pin management.
- [adc](microchip.pic12f6xx.adc.html) for the pin port management.


### PIC12F6xx family targets

  - PIC12F629
  - PIC12F675
  - PIC12F683
