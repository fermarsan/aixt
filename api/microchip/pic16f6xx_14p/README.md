## Description

`pic16f6xx_14p` is not a module by itself, but a namespace for the API modules of Microchip PIC16F6xx_14p target devices. Some of functions are shared by most of targets. 


### PIC16F6xx_14p modules

- [pic16](microchip.pic16.html) 
- [adc](microchip.pic16f6xx_14p.adc.html) 
- [pin_fn](microchip.pic16f6xx_14p.pin_fn.html) 
- [port](microchip.pic16f6xx_14p.port.html)


### PIC16F6xx_14p family targets

  - PIC16F630
  - PIC16F676
  - PIC16F684
