## Description

`pic16f88x` is not a module by itself, but a namespace for the API modules of Microchip PIC16f88x target devices. Some of functions are shared by most of targets. 


### PIC16F88x modules

- [pic16](microchip.pic16.html) 
- [pin_fn](microchip.pic16f88x.pin_fn.html) 
- [pin_fast](microchip.pic16f88x.pin_fast.html) 
- [port](microchip.pic16f88x.port.html)
- [adc](microchip.pic16f88x.adc.html)

### PIC16F88x family targets

  - PIC16F882
  - PIC16F883
  - PIC16F884
  - PIC16F886
  - PIC16F887
