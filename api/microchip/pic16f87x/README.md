## Description

`pic16f87x` is not a module by itself, but a namespace for the API modules of Microchip PIC16f87x target devices. Some of functions are shared by most of targets. 


### PIC16F87x modules

- [pic16](microchip.pic16.html) 
- [pin_fn](microchip.pic16f87x.pin_fn.html) 
- [pin_fast](microchip.pic16f87x.pin_fast.html) 
- [port](microchip.pic16f87x.port.html)
- [adc](microchip.pic16f87x.adc.html)

### PIC16F87x family targets

  - PIC16F873(A)
  - PIC16F874(A)
  - PIC16F876(A)
  - PIC16F877(A)
