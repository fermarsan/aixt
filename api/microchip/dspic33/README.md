## Description

The API for the PIC24_dsPIC33 core includes the modules:

- [pin_fn](microchip.pic24_dsPIC33.pin_fn.html) for the pin management.
- [pin_fast](microchip.pic24_dsPIC33.pin_fast.html) for the pin management (faster version).
- [port](microchip.pic24_dsPIC33.port.html) for the pin port management.
- [timer0](microchip.pic24_dsPIC33.timer0.html) for the timer0.
- [ext](microchip.pic24_dsPIC33.ext.html) for the external interrupt.
