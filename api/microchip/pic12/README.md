## Description

The API for the PIC12 includes the core modules and the additional ones.


### PIC12-core modules

- [pin_fn](Microchip.PIC12.pin_fn.html) for the pin management.
- [port](Microchip.PIC12.port.html) for the pin port management.
- [timer0](Microchip.PIC12.timer0.html) for the timer0.
- [ext](Microchip.PIC12.ext.html) for the external interrupt through GPIO0/INT pin.


### PIC12 additional modules

- [pwm](Microchip.PIC12_pwm.pwm.html) for the pin management.
- [adc](Microchip.PIC12_adc.adc.html) for the pin port management.
- [timer1](Microchip.PIC12_timer1.timer1.html) for the timer0.
- [timer2](Microchip.PIC12_timer2.timer2.html) for the timer0.


### Supported target devices

PIC12F6xx:

  - PIC12F629
  - PIC12F675
  - PIC12F683
