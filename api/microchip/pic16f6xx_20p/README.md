## Description

`pic16f6xx_20p` is not a module by itself, but a namespace for the API modules of Microchip PIC16F6xx_20p target devices. Some of functions are shared by most of targets. 


### PIC16F6xx_20p modules

- [pic16](microchip.pic16.html) 
- [pin_fn](microchip.pic16f6xx_20p.pin_fn.html) 
- [port](microchip.pic16f6xx_20p.port.html)

### PIC16F6xx_20p family targets

  - PIC16F685
  - PIC16F687
  - PIC16F689
  - PIC16F690
