## Description

`microchip` is not a module by itself, but a namespace for the Microchip devices API modules. Some of functions are shared by most of devices.

The API is split in several folders:

### Compiler related modules

- [xc8](microchip.xc8.html)
- [xc16](microchip.xc16.html)


### Device family related modules

- [PIC12](microchip.pic12.html)
- [PIC16](microchip.pic16.html)
- [PIC18](microchip.pic18.html)
- [PIC24](microchip.pic24.html)


### Supported targets
 
- [PIC12F6xx family](microchip.pic12f6xx.pwm.html)
- [PIC16F6xx_14p family](microchip.pic16f6xx_14p.pin_fn.html)
- [PIC16F6xx_18p family](microchip.pic16f6xx_18p.pin_fn.html) 
- [PIC16F6xx_20p family](microchip.pic16f6xx_20p.pin_fn.html)
- [PIC16F8x family](microchip.pic16f8x.pin_fn.html)
- [PIC16F87x family](microchip.pic16f87x.pin_fn.html)
- [PIC16F88x family](microchip.pic16f88x.pin_fn.html)
- [Exp16_PIC24](microchip.exp16_pic24.pin_fn.html)
