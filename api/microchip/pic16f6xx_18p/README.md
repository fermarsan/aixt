## Description

`pic16f6xx_18p` is not a module by itself, but a namespace for the API modules of Microchip PIC16F6xx_18p target devices. Some of functions are shared by most of targets. 


### PIC16F6xx_18p modules

- [pic16](microchip.pic16.html) 
- [pin_fn](microchip.pic16f6xx_18p.pin_fn.html) 
- [port](microchip.pic16f6xx_18p.port.html)


### PIC16F6xx_18p family targets

  - PIC16F627(A)
  - PIC16F628(A)
  - PIC16F648A
  - PIC16LF627A
  - PIC16LF628A
  - PIC16LF648A
