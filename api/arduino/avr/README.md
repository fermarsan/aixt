## Description

`avr` is not a module by itself, but a namespace for the Arduino AVR-based API. Some of functions are shared by most of devices.

The AVR API is split in several folders:

### Compiler "avr" related modules

- [pin](arduino.avr.pin.html)
- [pin_fn](arduino.avr.pin_fn.html)
- [port](arduino.avr.port.html)
- [spi](arduino.avr.spi.html)
