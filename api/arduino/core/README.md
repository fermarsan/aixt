## Description

`core` is not a module by itself, but a namespace for the Arduino core API modules. Some of functions are shared by most of devices.

The core API is split in several folders:

### Compiler "core" related modules

- [time](arduino.core.time.html)
- [ext](arduino.core.ext.html)
- [adc](arduino.core.adc.html)
- [adc_fn](arduino.core.adc_fn.html)
- [pin](arduino.core.pin.html)
- [pin_fn](arduino.core.pin_fn.html)
- [pwm](arduino.core.pwm.html)
- [pwm_fn](arduino.core.pwm_fn.html)
