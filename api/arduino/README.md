## Description

`arduino` is not a module by itself, but a namespace for the Arduino devices API modules. Some of functions are shared by most of devices.

The API is split in several folders:

### Compiler related modules

- [core](arduino.core.html)


### Device family related modules

- [uno](arduino.uno.html)
- [nano](arduino.nano.html)
- [mega](arduino.mega.html)


### Generic modules

- [avr](arduino.avr.html)
- [arm32](arduino.arm32.adc.html)
- [esp32](arduino.esp32.adc2.html)
- [i2c](arduino.i2c.i2c.html)
- [uart](arduino.uart.uart.html)
- [uart1](arduino.uart1.uart1.html)
- [uart2](arduino.uart2.uart2.html)
- [uart3](arduino.uart3.uart3.html)
- [uart4](arduino.uart4.uart4.html)
- [uart5](arduino.uart5.uart5.html)
