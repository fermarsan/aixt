# PIC16 core modules

The API for the PIC16 core includes the modules:

- [pin_fn](pin_fn.html) for the pin management.
- [pin_fast](pin_fast.html) for the pin management (faster version).
- [port](port.html) for the pin port management.
- [timer0](timer0.html) for the timer0.
- [ext](ext.html) for the external interrupt through RB0/INT pin.
- [extb](extb.html) for the external interrupt by changes on RB7-RB4 pins.
