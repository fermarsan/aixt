PIC16 core modules

The API for the PIC16 core includes the modules:

  • pin_fn for the pin management.
  • pin_fast for the pin management (faster version).
  • port for the pin port management.
  • timer0 for the timer0.
  • ext for the external interrupt through RB0/INT pin.
  • extb for the external interrupt by changes on RB7-RB4 pins.
